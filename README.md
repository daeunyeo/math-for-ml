# Mathematics for Machine Learning

Self-study notes based on Mathematics for Machine Learning
(Deisenroth, Faisal, Ong — Cambridge University Press, 2020).
Free PDF available at https://mml-book.com

## Purpose

Building mathematical foundations for ML research.
Focusing on conceptual understanding before diving into code,
where each topic is manually derived before being verified with NumPy.
